Flex布局教程

  • 白小霁
  • 3 Minutes
  • May 7, 2017

核心点：

container 、item 、 main/cross axis 、 main/cross start 、main/cross end 、 main size 、cross size

container （容器） 属性

  1. flex-direction
  2. flex-wrap
  3. flex-flow 是direction 和wrap的简写
  4. justify-content
  5. align-items
  6. align-content(用的教少)

item（项目）属性

  1. order
  2. flex-grow分剩余空间（如果有的话）
  3. flex-shrink定义了项目的缩小比例 item-width > container-width
  4. flex-basis 跟设置width或height一样效果
  5. flex 是 flex-grow 、 flex-shrink 、 flex-basis的简写
  6. align-self允许单个项目有与其他项目不一样的对齐方式

使用

  1. 手机布局 topbar + main+ tabs
  2. 产品列表
  3. PC页面布局
  4. 完美居中
