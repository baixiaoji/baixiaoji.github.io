学习vim之路

创建时间：2017/3/13
上次修改：2017/3/13

最近学习没有什么动力，于是开始决定学习vim的使用，主要的参考材料是如下：

  1. VIM的历史介绍
  2. COOLSHELL简明VIM练级攻略
  3. 寓教于乐-vim的游戏

因为看到酷壳的文章划分的很有意思，所以想以升级的方式去写。

首先，vim的一款非常难学的编辑器，却是不可思议的好用，当然也是提高逼格的一个好方法。

在开始学习以前，我需要给你一些警告（自勉）：

  • 学习vim在开始时是痛苦的。
  • 需要时间
  • 需要不断地练习，就像你学习一个乐器一样。
  • 不要期望你能在3天内把vim练得比别的编辑器更有效率。
  • 事实上，你需要2周时间的苦练，而不是3天。

方舟 —— 赤手空拳的活下来

现在，确保你的系统下面是有vim的，紧接下来就是想怎么在这个黑框中活下来。

  • 启动Vim后，vim在Normal模式下。
  • 如果想进入Insert模式（输入模式），请按下i键
  • 这时候，你就可以像记事本一样的编辑
  • 如果你想要回退到Normal模式，请按ESC
    这样下来，你就知道了如何在Normal和Insert模式的切换，但是想要存活下来还是要明白一些基本的操作。
  • i 👉 切换到Insert模式，按ESC回到Normal模式
  • x 👉 删除当前光标所在的一个字符，其内容存在剪贴板里
  • :wq 👉 写入（存档）+ 退出 (:w 写入 :q 退出)
  • dd 👉 删除当前行的所有字符，其内容存在剪贴板里
  • p 👉 粘贴剪切板
  • hjkl 光标的移动键（👈👇☝👉）
  • :help <command> 👉 显示相关的命令帮助（退出帮助请输入:q）

如果，能掌握上述的命令，就可以编辑文本了，像自勉中说的一样，需要练就能为下意识的状态，就能进入下一阶段了。

方舟 —— 小兵器时代

上述的命令，只能让你在vim方舟中，赤手空拳的活着，接下来的命令能让你，在方舟中迎来小兵器时代（注：所有的命令在Normal模式下面使用，如果不清楚当前模式，请狂按ESC几次）

  1. 开启各种插入模式

    • a 👉 在光标后插入
    • o 👉 在光标所在行后添加一行
    • O 👉 在光标所在的前添加一行
    • cw 👉 删除从光标所在位置到本行的结尾，其内容的存到剪贴板中

  2. 简单的移动光标

    • 0 👉 将光标移到行首
    • ^ 👉 将光标移到本行第一个不是blank字符的位置（blank字符就是空格、tab、换行、回车等）
    • $ 👉 将光标移到本行行尾
    • g_ 👉 将光标移到本行最后一个不是blank字符的位置。
    • /pattern 👉 搜索pattern的字符串（按n到下一个）

  3. 拷贝/粘贴

    • p/P 👉 粘贴分别是当前位置后/前
    • yy 👉 拷贝当行 === ddP

  4. Undo / Redo

    • u 👉 undo，撤回到上一次的操作
    • <Ctrl + r> 👉 redo，与undo相反

  5. 打开/保存/退出/改变文件

    • :e <path/to/file> 👉 打开一个文件
    • :w 👉 写入一个文件
    • :saveas <path/to/file> 👉 另存文件到 <path/to/file>
    • :x,ZZ或:wq 👉 保存并退出
    • :q! 👉 强制退出不保存
    • :bn和:bp 👉 可以同时打开多个文件，使用这两个命令可以切换下一个或上一个文件

能否进阶到小兵器时代，一定有花上时间来练习，一旦掌握，就可以做一些与其他编辑器能干的事了。下一次更新是笔者进阶到小兵器时代。

这是一个船票的故事

拿到这套由肯-福莱特《巨人的陨落》，目光只是被「平均3个通宵读完」这句话给吸引了，并且笔者尝试过一个
晚上真的读完了。整本书的描述的是一战前后几个家庭的兴衰变化。因为一直都图书馆读书，并且配上
rainy mood为背景音乐，心情真的很平静（墙裂推荐）。

当然说到书的作者，我是第一次听说，读完之后我才发现这是一个聚集艺术天赋的作家。为什么这么说了？首先，在第一本的导言部分
他说了这么一句话「很多作家只写能取悦他们自己的东西，并模模糊糊地希望这也能取悦别人。但我每写一页都在清醒地思考：读者会怎么想？
读者觉得这真的会发生吗？读者关心这些吗？读者想要知道接下来会发生什么吗？…」,不料的是，在书的过程当中，我真的以为这事一个真实发生的（即使每一本书的最后都有说明人物的真实性），当然在最后的致谢之前，老爷子还是跟读者讲解了他是如何让自己清楚的知道，哪些是历史那些是虚构的（那时候只有大写的佩服）。

一张船票引起的惨剧

读完整本书，一个道理不断的验证：「万事万物都是有所联系的」。首先说明我不会将很多联系，我只讲我看完正套书之后往前推，让我将这一系列事情的起因归结于这件事情————格雷戈里让出了他那一张通往美国的船票。这里的他则是他亲爱的弟弟。
如果没有看过故事的人一定会认为这是天经地义的事情，可为此抛弃了自己的理想，丢失了自己的初恋，还要养别人的孩子这三件事同时落在哥哥的身上，这种情况下如果你是哥哥，你会做什么决策。当然，书中的格雷戈里那一刻并不知道后面两件事情。这时候应该说点人设了，列夫是一个长相秀气，说话温柔的小伙子，换通俗的话便是列夫是一个撩妹高手。
这样列夫去了美国了吗？当然后来是去了，还美国有了自己的家庭。起初，他拿着他哥船票，由于被维亚洛夫家骗了到了一个小岛上做起了苦力。当然他都会通过一些小伎俩去赚钱，无论在船上还是在小岛上一贯如此，他同样受到了副作用，被一个曾经的「伙伴」出卖了，以至于跑路到了美国。到了他哥梦想的国度，他有点良心有时候记得对他哥的承诺，赚钱寄回给哥哥来美国。凭借着他浑身是胆的品格，被维亚洛夫的大家长看上（没错就是之前骗他做苦力的这个家族），做了大家长的司机。说过他的掩藏技能可是撩妹呀，结果撩上了大家长的女儿，居然还怀上了，这可是他的第二个女人。当然他并不是故事里的「大仲马」，但一定是「小仲马」。不巧的是，大家长在不知情的情况下，让女儿与美国总统的幕僚格斯订婚了。但是得知之后，大家长还是一个传统人士，依旧是
让女儿与列夫结婚了。婚后，依旧不肯放弃这一整片森林的列夫，发生了若干的婚外情，不知道为什么大家长都会逮个正着，之后逼他去当兵，即使当兵还不改本性噢！当之无愧的「小仲马」。当然一山不容二虎，最后大家长还是死在了和列夫的争执当中。
当然，故事中列夫还是赚到了钱给他哥哥，但是他哥哥并没有去美国。那时候哥哥也算的上一位重要的人物吧，是辅佐列宁的人哦。想知道为什么这时候他哥哥不去美国了吗？就不说，微笑脸。

爱你，我愿意等；爱你，我现在才知道

故事当中描述了很多情侣，其中有婚外情很多，当然还有美好的爱情呀。
沃尔特与茉黛，分别出生于德国和英国。你还记得书的名字吗？其中的巨人我原本认为只有英国，但现在我觉得应该是参与战争后都受损的国家。第一本书，在他俩秘密结婚结束，记录的他们第二次同房的时间可是而是二战期间噢，最后德国失败了，他们到了当初预定好的地方开始之后生活。又该讲解人设了，沃尔父亲（外交官）计划好了沃尔特的人生之路无疑是当上一个比他老爹更厉害的外交官咯。茉黛勋爵是「大仲马」菲茨伯爵的妹妹。中途，沃尔特被父母逼迫去和他妹妹的密友莫妮卡结婚，当然莫妮卡机智的发现了沃尔特结婚的秘密，失落却很理智的说了一句「你是我见过的最好的男人。我不会做任何伤害你的事。我永远不会吐露一个字」。看出莫妮卡真的很爱沃尔特，只不过验证了书中莫妮卡说的那句话「但她赶我前面发现了你，就是这样。」，一个看的很透的女性，最后的「就是这样」意味深长，她一定要找一个是她幸福的人(因为书中并没有莫妮卡的结局)。当然，在两国交战的时候，他两秘密有信件来往，有一次就是格斯当了他们爱的邮差。而茉黛总是担心沃尔特忘记了她，可能像他哥一样沾花惹草，有时候反倒希望沃尔特在战争中受伤。不过爱情还是经得起时间的考验的，最后她放下了自己的身份在一个酒馆当钢琴师这一点足以说明。

格斯与罗莎：我等到了你，真是我的幸运。一位美国总统的幕僚，一位则是无政府报社的记者。再被列夫抢走未婚妻的之前格斯就和罗莎认识。在格斯说他喜欢奥尔加（大家长的女儿）时，罗莎之前是有话对格斯讲的，但那一刻我感受了到了罗莎仿佛在哭，因为她一直都很仰慕格斯，但是她不说，而他又陷入自己的情欲当中无法自拔。直到最后格斯说了一句我感触很深的话「我想也许我爱上你已经有了相当长的时间了，可自己一直都不知道」，当然罗莎并没有回应他，这里有了一种谁叫你上次让我伤心这次我就叼你胃口的感觉。撒了一波的狗粮呀。通过描述我还是认为罗莎也是一个美女，即使身体上有一定的缺陷。今中午，在知乎上看到了一句话「每一个人在真正遇到喜欢的人之前，我们评判对方的标准是脸，并不是在意他/她的品格…」(大概是这个意思，微笑脸)。
当然还有其他的情侣档，为什么不写？因为不知道怎么理思路，都会想到「大仲马」菲茨赫伯特伯爵，为什么对他有成见！因为在他身上看到的只是男性只会下半身思考的动物。当然他也是一个撩妹高手，在他毁容兼残疾的情况下还是那么会撩。对他很成见的另一个原因就是笔者还是单身！！！

当然书中有写到性爱，具体没数感觉蛮多的。而且描写的还是蛮干才烈火的，茉黛说了一句「来吧，时间不多了」，时间在一战期间她和沃尔特的见面说了的一句话，地点大家自己想象咯。书还是蛮不错的，如果真的可以耐下心来读的话，一天其实都可以读完的。如果对历史感兴趣的同学真的推荐读一下，不知道会不会跟我一样能够有一种真假难分的感觉。
ps：当然这本书有很多书评都写了这本的书的一些大概故事情节，并有一些稍微透露了情节。我在周五晚上了通宵看完「蝴蝶效应」这部电影，所以有了写这篇书评的稍稍灵感。
